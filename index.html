    Tech Skills

    개발의 시작은 C의 기초로 탄탄하게 시작하였습니다. 이후 Java, C++, Python, Web 언어 등 여러 언어를 배움과 함께 자료구조와 알고리즘에 대해 공부했습니다. IoT 개론을 통해 컴퓨터 구조부터 용어를 학습하였고, 전기전자논리회로를 통해 (간단한) 회로 설계와 HW 회로 제작을 실습했습니다. 이를 바탕으로 회로의 센서를 이용하여 임베디드 시스템, IoT 플랫폼 과목에서 Arduino, RaspberryPi를 활용하여 IoT 서비스을 개발했습니다. 리눅스 프로그래밍을 통해 Windows가 아닌 다른 환경에서의 개발을 할 수 있게 되었으며, 컴퓨터 네트워크, 네트워크프로그래밍 과목을 통해 Bluetooth, Wi-Fi, Socket, API 등 여러 통신 개발도 가능해졌습니다. 그리고 Network 연구실의 연구학생(2018.03-2020.08)으로 배운 기술들을 아울러 IoT 시스템이 적용된 스마트 플러그, 공용 냉장고 內 도난 방지 시스템 등의 프로젝트를 진행했습니다. 이를 바탕으로 생각하는 아이템에 대해 규모에 따라 걸리는 시간은 다르지만 혼자서도 IoT 프로젝트를 개발할 수 있습니다.

    프로젝트 관리툴로는 Github, Slack, Notion, Google Drive 등을 사용하였습니다.
